# Top 10 nejpoužívanějších pluginů pro #obsidian

| #   | Plugin          | Popis                                                              |
| --- | --------------- | ------------------------------------------------------------------ |
| 1   | Dataview        | Dotazy nad poznámkami, tabulky, databázové pohledy na obsah vaultu |
| 2   | Templater       | Pokročilé šablony s podporou JavaScriptu                           |
| 3   | Calendar        | Kalendář a integrace s Daily Notes                                 |
| 4   | Tasks           | Správa úkolů, filtrování a přehledy                                |
| 5   | Excalidraw      | Kreslení diagramů a myšlenkových map přímo v Obsidianu             |
| 6   | Kanban          | Kanban tabule pro řízení projektů                                  |
| 7   | QuickAdd        | Automatizace workflow a rychlé vytváření poznámek                  |
| 8   | Advanced Tables | Pohodlná editace Markdown tabulek                                  |
| 9   | Style Settings  | Nastavení vzhledu kompatibilních témat                             |
| 10  | Commander       | Vlastní tlačítka, příkazy a úpravy uživatelského rozhraní          |
